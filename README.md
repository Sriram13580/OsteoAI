# 🦴 OsteoAI — AI-Powered Osteoporosis Detection System

> **Proof-of-Concept** AI Healthcare Web Application for early detection of osteoporosis using X-ray image analysis and clinical risk prediction, integrated with an intelligent medical AI chatbot.

---

## 🚀 Quick Start (30 seconds)

### Option A — PowerShell Script (Recommended)
```powershell
cd osteoporosis-app
.\start.ps1
```
Then open **http://localhost:5173** in your browser.

### Option B — Manual Start

**Terminal 1 — Backend:**
```bash
cd backend
node mock_server.js
```

**Terminal 2 — Frontend:**
```bash
cd frontend
npm run dev
```

Open **http://localhost:5173** 🎉

## 🧠 AI Architecture

### Hybrid Decision Engine
```
Final Risk Score = (0.6 × CNN Probability) + (0.4 × Clinical Risk Score)
```

| Score Range | Stage | Risk Level |
|-------------|-------|------------|
| 0 – 39      | Normal | 🟢 Low |
| 40 – 69     | Osteopenia | 🟡 Moderate |
| 70 – 100    | Osteoporosis | 🔴 High |

### CNN Image Model
- Architecture: **MobileNetV2** + Global Average Pooling + Softmax
- Input: 224×224 RGB X-ray images
- Output: Normal / Osteopenia / Osteoporosis + confidence %

### Clinical Risk Model
- Algorithm: **Gradient Boosting Classifier** (scikit-learn)
- Features: Age, Gender, BMI, Family History, Previous Fracture, Lifestyle Risk, Calcium Intake, Smoking, Alcohol
- Output: Clinical risk score (0–100)

## 📦 Dataset Information

This demo uses **synthetic AI-generated data**:

- **X-Ray Images**: Procedurally generated bone-density-mimicking images (180 samples)
  - Normal: bright, dense bone structure
  - Osteopenia: moderate density with porosity patterns
  - Osteoporosis: low density with high porosity
- **Clinical Dataset**: 1,000 rows of synthetic patient records (NumPy + Pandas)

> For production use: Replace with validated Kaggle datasets such as:
> - [Bone Fracture Detection](https://www.kaggle.com/datasets)
> - [Osteoporosis Risk Dataset](https://www.kaggle.com/datasets)
